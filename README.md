# Visual-Hallucination-Detection-in-AI-Generated-images
This project focuses on detecting visual hallucinations in images generated by AI models, I have used 3 type of dataset for this project like Real fame images FFHQ Dataset, 
Real object  COCO dataset  and i will generate the ai images using stable diffusion method can generate images from text prompts. Sometimes created images that contain visual error. 
These error are called visula hallucination.

The main aim of my project is detecting the hallucination images automatically before user see the images, because everyone use the ai images the hallucinated images reduce the trust of 
users. To solve the problem i have used CNN model i will give the result like label on hallucinated images.and i will get the result of accuracy 91.25
